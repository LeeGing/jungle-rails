## Dependencies

BCrypt
Faker
Money-Rails
Puma
PostgresSQL 9.x
Stripe
Rails 4.2
RMagick


## Getting Started

Install all dependencies (using gem install command).
Run the server using bin/rails s -b 0.0.0.0 command.
Go to http://localhost:3000/.
